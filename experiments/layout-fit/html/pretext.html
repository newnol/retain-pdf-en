Pretext PoC

Using the same `fixtures` to compare DOM measurement and `@chenglou/pretext`. This page focuses only on block-level text: same text, same width, same line height, comparing `height`, `lineCount`, `maxLineWidth`, and line-by-line text between the two measurement paths.
