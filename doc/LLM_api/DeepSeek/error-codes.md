When calling the DeepSeek API, you may encounter the following errors. Here are the causes and solutions for these errors.

Error Code	Description
400 - Bad Request	Cause: Malformed request body
Solution: Please modify the request body according to the error message
401 - Authentication Failed	Cause: Incorrect API key, authentication failed
Solution: Please check if your API key is correct. If you don't have an API key, please create one first
402 - Insufficient Balance	Cause: Insufficient account balance
Solution: Please check your account balance and go to the Top-up page to recharge
422 - Invalid Parameter	Cause: Invalid parameters in the request body
Solution: Please modify the relevant parameters according to the error message
429 - Rate Limit Reached	Cause: Request rate (TPM or RPM) has reached the limit
Solution: Please plan your request rate reasonably
500 - Server Error	Cause: Internal server error
Solution: Please wait and retry. If the problem persists, please contact us for resolution
503 - Server Busy	Cause: Server is overloaded
Solution: Please retry your request later
